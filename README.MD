# Messier Finder

Messier finder is an application developed by team two, csc 380 25F of SUNY Oswego. This app serves the purpose as a database for Messier Catalogue objects. The program functions through the use of user-entered parameters based on observatory and telescope settings.

## Use

to use this program:

1. Download the appropriate package for your operative system: Windows/Linux
2. Execute the program.
3. Set up environment by entering observatory, telescope, and obstruction settings.
4. The program has a UI, any filters, or search may be conducted, you may also browse through the entire catalogue.

## technical

A JRE is not provided with the code, one must be made, to do so, please download the javafx25 Jmods, and use jlink version 25.0.1 in order to generate the appropriate runtime. Packages bundled with the jre using jpackage are provided.
